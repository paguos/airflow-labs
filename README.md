# airflow-labs
Playground to play with Apache Airflow

## Requirements

- pipenv
- Python 3.11

## Airflow 

### Setup

1. Initialize airflow by running (only once):
```sh
make airflow-init
```

2. Start the webserver:
```sh
make airflow-webserver
```

3. Start the scheduler:
```sh
make airflow-scheduler
```
